# Knowledge Base

Place your PDF documents in this directory. The RAG chatbot will read and process PDFs from here.

## Usage

1. Add your PDF files to this directory
2. Run the Streamlit application: `streamlit run rag.py`
3. Select the PDF you want to query from the dropdown menu
4. Ask questions about the selected PDF

## Example

```
knowledge_base/
├── document1.pdf
├── document2.pdf
└── research_paper.pdf
```
